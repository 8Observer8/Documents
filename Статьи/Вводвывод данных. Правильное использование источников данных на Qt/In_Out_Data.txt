[b]Текстовый файл[/b]

Описание:
- программа читает массив целых чисел из файла "input.txt". Каждое число не должно выходить за диапазон [-100, 101]
- сохраняет этот массив в файл "output.txt"

Программа выводит следующие тексты ошибок:
- "Error: unable to open the file input.txt" - если не удалось открыть файл input.txt
- "Error: unable to read the file input.txt at the line 3" - если, к примеру, на третьей строке файла расположен какой-либо символ вместо целого числа
- "Error: unable to write the file output.txt" - если диск переполнен и невозможно записать данные в файл output.txt
- "Error: empty argument" - если данные отсутствуют
- "Error: the argument 101 doesn't hit to the range [-100, 100]" - если число не попадает в допустимый диапазон
