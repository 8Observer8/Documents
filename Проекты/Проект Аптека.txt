Проект Аптека

Привет!

Решил реализовать проект Аптека, под вашим чутким руководством, дорогие форумчане!

На первом этапе приложение будет консольным.

Пользователь может вводить команды:

add - добавить лекарство
show - показать список лекарств
del - удалить последнее лекарство
sort -name - сортировать по ключу
sort -type - сортировать по типу
sort -price - сортироваить по цене
sort -num - сортировать по серийному номеру

Содержание:
Запись в бинарный файл
Запись в Json файл
Запись в XML файл
Запись в базу данных SQLite
Наследование
Сортировка по заданному полю
